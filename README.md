# LimeNit Logo Generator

A professional, free online tool for creating stunning SVG logos instantly.

## Features

- 18 industry-specific themes
- 5 complexity levels
- Multiple base shapes and frames
- 8 color palettes + custom colors
- Real-time preview
- SVG export and download

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No frameworks or dependencies
- Optimized for Vercel deployment

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Start designing!
